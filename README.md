# Omazon

It is an Amozon-like storefront with MySQL database backend.
The Omazon will take in orders from customers and deplete stock from the store's inventory.

### Set up

1. Clone the repo

2. `npm install`

3. Run  `node omazon.js`

4. Thats it! Bam!
